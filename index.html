  2. Explain how many different types of lists we have in html
    3. different types of lists
    There are three list types in HTML: unordered list — used to group a set of related items in no particular order. ordered list — used to group a set of related items in a specific order. description list — used to display name/value pairs such as terms and definitions.
  3. What are semantic tags ?
    4. Semantic HTML tags are tags that define the meaning of the content they contain. For example, tags like header, article, and footer are semantic HTML tags. They clearly indicate the role of the content they contain.
